# 아바타 설정

## 액터 추가 및 아바타 업로드
### 액터 추가
![AvatarSetImage1](/img/Page_AvatarSettings/AvatarSettings_1.png) <br/>
VIRDY는 사용자의 캐릭터를 <span class="highlight_text">**액터**</span>라 칭하고 있습니다. <br/>
최초 접속시 액터를 생성하여야 합니다. 생성된 액터는 <span class="highlight_text">**상단 탭**</span>에 구분됩니다.

### 아바타 추가
![AvatarSetImage2](/img/Page_AvatarSettings/AvatarSettings_2.png) <br/>
우측 액터 패널에서 <span class="highlight_text">**아바타 추가**</span>를 클릭하여 아바타를 업로드 할 수 있습니다. <br/>
**``vrm``** 혹은 **``vsf``** 파일을 현재 지원중에 있습니다.

<img src="/VIRDY-Docs/img/Page_AvatarSettings/AvatarSettings_3.png" alt="AvatarSetImage3" width="50%;"/>

아바타 업로드가 완료되면 해당 패널에 <span class="highlight_text">**카드 형태**</span>로 저장됩니다. <br/>
아바타를 선택하여 액터에 사용하고자 하는 대표 아바타를 설정해주세요.

### 액터 이름 설정
<img src="/VIRDY-Docs/img/Page_AvatarSettings/AvatarSettings_4.png" alt="AvatarSetImage3" width="50%;"/>
액터별로 이름을 설정할 수 있습니다. <br/>
월드 접속 후에는 액터 이름이 함께 표기되기에 별도 이름을 지정해주시는 편이 좋습니다.

### 아바타 트래킹 설정
<img src="/VIRDY-Docs/img/Page_AvatarSettings/AvatarSettings_5.png" alt="AvatarSetImage3"/>
좌측 사이드바의 아이콘을 클릭하여 <span class="highlight_text">**Facial / Body / Hand Tracking**</span>을 설정할 수 있습니다.

## 사용 가능한 기기

<span class="highlight_text">**VIRDY**</span>에서는 아래 모션캡처 / 페이셜 장비를 지원합니다.

### Facial Tracking

- **```iFacialMocap```**
- **```FaceMotion3D```**
- **```VTubeStudio```**
- **```OpenSeeFace```**

### Motion Tracking

- **```Xsens```**
- **```Vicon```**
- **```SteamVR```**
- **```VMC```**
