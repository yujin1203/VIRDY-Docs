# 아바타 세부설정

## 아바타 오브젝트 On/Off
![AvatarAdvancedSettings](/img/Page_AvatarSettings/AvatarAdvancedSettings_1.png) <br/>
![AvatarAdvancedSettings](/img/Page_AvatarSettings/AvatarAdvancedSettings_2.png) <br/>
아바타 오브젝트의 Toggle 기능은 <span class="highlight_text">**'아바타 세부설정'**</span> 탭에서 조작할 수 있습니다. <br/>
끄고 싶은 오브젝트를 안보이게 하거나, 여러개의 의상과 헤어를 겹쳐두고 원하는 형태로 On/Off를 할 수 있습니다.

:::caution CAUTION
의상, 헤어 등을 여러개 겹쳐두고 On/Off를 하더라도 메모리상으로는 모두 적용되어 빌드됩니다. <br/>
과도한 사용은 <span class="highlight_text">**메모리 부하**</span> 및 <span class="highlight_text">**월드 내 유저들의 프레임**</span>에도 영향이 갈 수 있으므로, 적절히 사용하시기 바랍니다.
:::

## 표정 단축키 설정
![AvatarAdvancedSettings](/img/Page_AvatarSettings/AvatarAdvancedSettings_3.png) <br/>
표정 단축키 설정 탭에 들어가 <span class="highlight_text">**Blendshape 표정 단축키**</span>를 설정할 수 있습니다. <br/>
설정 더보기란을 눌러 더 자세한 옵션을 설정할 수 있습니다.
