# UI

## 기본 UI

![InGameUI](/img/Page_UI/InGame_Main_1.png) <br/>
<span class="highlight_text">**ESC**</span>를 눌러 인게임 내 기본 UI를 **<span class="highlight_text">On/Off</span>** 할 수 있습니다.

![InGameUI](/img/Page_UI/InGame_Main_2.png) <br/>
좌측 상단에서 확인할 수 있는 정보는 아래와 같습니다.

- **접속자 목록**
  - 현재 월드에 접속한 유저 목록을 확인할 수 있습니다.
- **아바타 / 월드 탭 이동**
  - 아바타/월드 버튼을 눌러 설정 화면으로 이동할 수 있습니다. <br/>
    월드에서 나가지 않습니다.
- **월드 코드 확인 및 복사**
  - 열쇠 아이콘을 눌러 월드 코드를 확인 및 복사할 수 있습니다.
- **월드 나가기**
  - 나가기 버튼을 눌러 해당 월드에서 퇴장할 수 있습니다.

## 주요 조작 버튼

![InGameUI](/img/Page_UI/InGame_Main_3.png) <br/>
VIRDY 조작에 대한 **<span class="highlight_text">주요 기능</span>** 버튼입니다.

- **카메라**
  - 카메라 설정 및 조작에 관한 기능을 담고 있습니다.
  - 단축키는 **'F1'** 입니다.
- **아바타**
  - 아바타 설정 및 조작, 변경 등에 대한 기능을 담고 있습니다.
  - 단축키는 **'F2'** 입니다.
- **라이트 (포스트 프로세싱)**
  - 라이트 및 포스트 프로세싱 값을 조절할 수 있는 기능을 담고 있습니다.
  - 단축키는 **'F3'** 입니다.
- **실험 기능**
  - 여러가지 실험적인 기능을 담고있습니다.
  - 단축키는 **'F4'** 입니다.

## 카메라 / 아바타 조작 전환

![InGameUI](/img/Page_UI/InGame_Main_4.png) <br/>
<span class="highlight_text">**카메라 / 아바타 조작**</span>을 전환하는 버튼입니다. <br/>
클릭시 스위칭되며, 단축키는 **<span class="highlight_text">'T'</span>** 입니다.
