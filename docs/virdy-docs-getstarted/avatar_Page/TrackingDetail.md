# 트래킹 디테일 설정

## 트래킹 세부 설정
![AvatarSettingsImage9](/img/Page_AvatarSettings/Tracking_Details_1.png) <br/>
사이드바 트래킹 세부 설정 아이콘을 눌러 여러가지 기능을 선택적으로 사용할 수 있습니다.

![AvatarSettingsImage9](/img/Page_AvatarSettings/Tracking_Details_2.png) <br/>
- **거울모드**
  - 모션 데이터를 반전시켜 적용합니다.
- **모션 보간**
  - 값이 높을 수록 모션이 천천히 따라 움직이고, 낮을 수록 지연 시간 없이 적용됩니다. <br/> 트래킹의 떨림이 심하거나 부드럽지 않을 때 적절히 조절하면 좋습니다.
- **화면에 스팀VR 트래커 표시**
  - SteamVR을 사용할 때 로비 화면에 트래커 아이콘이 표시됩니다. 이에 대한 불투명도 값 입니다.
