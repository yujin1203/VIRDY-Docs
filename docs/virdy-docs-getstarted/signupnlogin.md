# 회원가입 및 로그인

## 로그인
![SignupImage1](/img/Page_Signup/Sign_1.png) <br/>
VIRDY 로그인 화면입니다. 등록한 <span class="highlight_text">**이메일 혹은 닉네임**</span>과 비밀번호를 입력하여 로그인 할 수 있습니다. <br/>
최초 사용시에는 <span className="highlight_text">**회원가입**</span>를 눌러 회원가입을 진행해주세요.

## 회원가입

![SignupImage2](/img/Page_Signup/Sign_2.png) <br/>
본인이 크리에이터일 경우 <span class="highlight">**크리에이터**</span>를, 매니저(스태프)의 경우 <span class="highlight">**매니저**</span>를 선택해주세요.
:::tip 크리에이터? 매니저?

크리에이터 / 매니저 선택은 **사용자 구분**을 위한 목적입니다. <br/>
기능성 차이는 없지만 본인의 역할을 잘 체크하여 주시기 바랍니다.

:::

![SignupImage3](/img/Page_Signup/Sign_3.png) <br/>
정보를 입력 후 <span class="highlight">**이용약관 및 개인정보 수집 이용 동의**</span>를 체크하여 주세요.

이후 회원가입이 완료되면, 입력한 이메일에 자동으로 <span class="highlight_text">**인증 메일**</span>이 발송됩니다. <br/>
해당 메일을 통해 인증이 완료되면 로그인 할 수 있습니다. <br/>

:::danger 메일이 없어요 !

만약 이메일이 도착하지 않았다면 **스팸메일함** 등 다른 메일함을 확인해주세요.

:::

## 비밀번호 찾기

<img src="/VIRDY-Docs/img/Page_Signup/Sign_4.png" alt="Signup4" width="60%;"/>
<span class="highlight_text">**비밀번호를 잊으셨나요?**</span> 를 클릭하여 비밀번호를 재설정 할 수 있습니다.
<img src="/VIRDY-Docs/img/Page_Signup/Sign_5.png" alt="Signup4" width="60%;"/>
등록 당시 사용한 이메일, 닉네임을 입력 후 <span class="highlight_text">**비밀번호 재설정**</span>을 눌러주세요. <br/>
해당 이메일로 비밀번호 재설정 메일을 보내드립니다.
