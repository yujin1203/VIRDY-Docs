# VIRDY 셰이더 개요

VIRDY는 <span class="highlight_text">**Unity URP Forward+**</span> 환경을 사용합니다. <br/>
해당 렌더러 환경을 지원하는 셰이더는 모두 사용 가능합니다.

:::caution 셰이더 스크립트
<span class="highlight_text">**NiloToon**</span>, <span class="highlight_text">**PotaToon**</span>과 같이 <span class="highlight_text">**특정 Script의 런타임 작동**</span>이 필요한 셰이더는 <br/>
VIRDY 내에 해당 스크립트가 포함되어야 합니다. 현재 지원중인 셰이더는 아래 목록에서 확인할 수 있습니다. <br/>
만약, 사용하고 싶은 셰이더가 지원되지 않는 경우에는 별도 문의 바랍니다.
:::
